# Codexにタスクを渡すためのプロンプト

あなたはChrome拡張の実装担当です。以下の要件を満たす「YouTube-Loop-Marker」を実装してください。

- YouTubeの再生中にA地点/B地点を設定し、A〜B区間をループ再生できる
- A/B地点の設定は必ずマウス操作のみで行う（ショートカットで設定は禁止）
- A/B地点へのジャンプはショートカットキーで可能にする（mac向けにCommand/Optionを使用）
- ループON/OFF、A/Bの表示を含む軽量UIをYouTubeプレイヤー上に追加する
- Manifest V3構成で、content script中心に実装
- `timeupdate`でループ制御を行う
- 最初に軽く設計方針を書き、その後コードを書く
- 最後に、拡張の読み込み・実行方法を説明する
- 会話/ドキュメントはすべて日本語で出力する
